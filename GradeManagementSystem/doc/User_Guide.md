# User Guide – Grade Management System

## Menu Options
1. Add Student Marks
2. View All Students
3. Calculate Averages
4. Find Top Performers
5. Generate Performance Report
6. Exit

## Add Student Marks
- Enter student name
- Enter marks for 5 subjects (0–100)
- Average, grade, and pass status calculated automatically

## View All Students
- Displays subject-wise marks in tabular format

## Calculate Averages
- Shows average, grade, and pass/fail status

## Top Performers
- Ranks students based on average marks

## Performance Report
- Subject averages
- Student ranking
- Grade distribution
